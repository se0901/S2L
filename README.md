# DevelopManagement

## 環境構築（簡易版）
### 前提条件　

以下のツールがインストールされていること
- node.js
  - https://nodejs.org/ja/download/
- Vue CLI
  - https://cli.vuejs.org/#getting-started
  - `npm install -g @vue/cli`


### 構築手順
#### プロジェクトルートで以下のコマンドを実行
1. パッケージのインストール
```
npm ci
```
2. プロジェクト立ち上げ
```
npm run serve
```
⇒ ブラウザで http://localhost:8080 にアクセス
